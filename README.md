# 🧠 Braincode | L'Assistant IA Copilote

> Propulsé par **Mekeirish** • Partie de *The Code Suite*

Braincode est l'intelligence centrale de ton workflow. C'est une interface de discussion hautement optimisée conçue pour dialoguer avec les modèles d'IA, conceptualiser des architectures et générer les briques logicielles de tes futures applications.

## 🔮 Interface & Design
*   **Charte Graphique :** `The Code Suite - Strict` (Matte Cyber & Soft UI).
*   **Coloris :** Bulles utilisateur détourées en Violet Électrique, réponses IA sur fond gris mat (`#16161a`). Indicateurs d'état et reflets de boutons de copie en Jaune Cyber.
*   **Layout :** Console de dialogue épurée avec barre de saisie ancrée en bas en flou d'arrière-plan (`backdrop-blur`).

## 🚀 Fonctionnalités Clés
*   **Flux de Discussion Fluide :** Interface réactive dédiée au streaming de code sans latence visuelle.
*   **Composant Code Intégré :** Isolation des blocs de code générés par l'IA avec bouton de copie instantanée intégré.
*   **Saisie Confortable :** Zone d'input large et responsive, optimisée pour l'envoi de prompts complexes et textuels.
